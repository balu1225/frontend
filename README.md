# Frontend Practice Projects

This repo hosts my practice solutions from [roadmap.sh](https://roadmap.sh/). Each subdirectory contains an independent HTML/CSS project.

## Projects

- `personal_portfolio`: implementation of the Portfolio Website brief — https://roadmap.sh/projects/portfolio-website
- `Single-Page CV`: single-page resume layout — https://roadmap.sh/projects/single-page-cv
- `Basic_Html_Website`: foundational multi-section page — https://roadmap.sh/projects/basic-html-website

Open a project folder and load `index.html` in any browser to preview it.
