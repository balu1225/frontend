# My Portfolio - Basic HTML Website

A clean and professional portfolio website showcasing my skills, projects, work experience, and education.

## Project Overview

This is a simple HTML and CSS-based portfolio website for Hemanth Balakrishna Nuthalapati. It features a responsive design with sections for projects, work experience, education, and reviews.

## Features

- **Responsive Navigation Bar** - Easy navigation between different sections
- **Home/Intro Section** - Introduction and key skills display
- **Projects Section** - Showcase of professional projects
- **Work Experience** - Professional background and experience
- **Education** - Academic qualifications
- **Reviews Section** - Testimonials and feedback

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Styling and responsive design
- **Vanilla JavaScript** - Interactive elements (if any)

## Skills Highlighted

- Java
- Spring Boot
- Node.js
- Angular
- React
- AWS
- PostgreSQL
- MongoDB

## Files

- `index.html` - Main HTML file with all content sections
- `styles.css` - CSS styling for the website

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the navbar links
3. Explore projects, experience, education, and reviews

Simply clone or download this repository and open the `index.html` file in your preferred web browser. No build tools or dependencies required!
